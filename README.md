# Second Half Project

A modern, responsive website for the Second Half Project - a 12-week one-on-one coaching program designed to restore stable energy, improve metabolic health, and support sustainable weight loss.

## Project Overview

The Second Half Project is about repairing your metabolism and transforming your body through a structured, personalized coaching approach. This website showcases the program and makes it easy for visitors to learn more and apply for coaching.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Hero Section** - Eye-catching headline and call-to-action
- **Three Pillars** - Interactive cards showcasing the core approach: Restore, Stabilise, Transform
- **Approach Section** - Information about the personalized coaching process
- **Contact Footer** - Easy access to phone, email, and social media links with icon buttons

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with flexbox and grid layout
- **SVG Icons** - Lightweight, scalable icons for contact methods and social media
- **Responsive Design** - Mobile-first approach with media queries

## File Descriptions

### index.html
Contains the complete structure of the website including:
- Navigation bar with logo and menu
- Hero section with main messaging
- Three pillars section showcasing the core approach
- Approach section with key benefits
- Footer with contact information and social media links

### style.css
Includes styling for:
- Global styles and typography
- Navigation bar with sticky positioning
- Hero section with gradient background
- Pillar cards with hover effects
- Responsive design for mobile devices (under 768px)
- Footer styling with flexbox layout
