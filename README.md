# Second Half Project

A modern, responsive website for the Second Half Project - a 12-week personalized one-on-one coaching program designed to restore metabolic health and support sustainable body transformation.

## Project Overview

The Second Half Project is about helping you reclaim your health through a structured, highly personalized coaching experience. This website provides information about the program, the coaching approach, and makes it easy for interested individuals to apply.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Multi-page Navigation** - Home, About, Program, and Apply pages
- **Hero Section** - Compelling introduction with Learn More button linking to About page
- **Three Pillars** - Cards explaining the core coaching approach: Relationship, Mindset, and Accountability
- **Approach Section** - Detailed information about the personalized coaching process
- **Why Section** - Clear messaging about the unique value proposition
- **Who Is This For Section** - Lists of common struggles and benefits
- **What's Included Section** - Details of program components with two-column layout
- **FAQ Section** - Expandable questions and answers about the program
- **Responsive Footer** - Contact information, social media icons, and links

## Pages

- **index.html** - Home page with hero section and program overview
- **about.html** - About the Second Half Project and who it's for
- **program.html** - Program details page, what's included, and FAQs
- **apply.html** - Application page with call-to-action for the Google Form

## Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Modern styling with flexbox and grid layout
- **SVG Icons** - Scalable icons for contact methods and social media
- **Responsive Design** - Mobile-first approach with media queries (mobile breakpoint at 768px)
- **JavaScript** - FAQ accordion functionality
